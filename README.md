# 2026 World Cup Predictions Tool

An interactive web tool predicting World Cup outcomes using Bayesian Neural Networks.

## Features
- Match predictions (Win %, Goals, Scorelines)
- Half-by-half goal timing
- Player performance predictions
- Interactive tournament brackets
- "What-if" player injury analysis
- Historical accuracy tracking

## How to Use
Visit: https://tomclegg123.github.io/2026-World-Cup/

## Technology
- **Model**: Bayesian Neural Network + Elo Ensemble
- **Training**: TensorFlow (Python)
- **Inference**: TensorFlow.js (React)
- **Hosting**: GitHub Pages
